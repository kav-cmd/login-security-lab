🔍 Security Vulnerabilities
❶Password Field
MD5 hashing — trivially reversible, no salt
Fix: Use bcrypt or argon2
❷Login Endpoint
No rate limiting by default
Fix: Enable rate limiting in dashboard
❸Error Messages
Username enumeration — "Invalid username" vs "Wrong password"
Fix: Return identical errors
❹Session Cookie
Predictable dev-secret-key signs tokens
Fix: Set a strong SECRET_KEY in env
❺Login Form
No CSRF token
Fix: Use Flask-WTF csrf_token()
❻No MFA
Single-factor only
Fix: Enable MFA in dashboard
❼Math CAPTCHA
Arithmetic challenges are machine-solvable in <1ms. A script can parse the question from the HTML, compute the answer, and submit — indistinguishable from a human.
Fix: Use a proof-of-work CAPTCHA (Cloudflare Turnstile, hCaptcha) or behavioural analysis — challenges that cannot be solved by reading the DOM.
