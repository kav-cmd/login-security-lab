## 🎤 THE 3 THINGS YOUR AUDIENCE SHOULD LEARN

After your demo, your audience should understand:

1. **Attacks are real and automated**
   - Takes seconds without protection
   - Millions of attacks happen daily
   - Everyone is a target

2. **Single defenses aren't enough**
   - Each has weaknesses/bypasses
   - Rate limit → attacker waits
   - Lockout → attacker tries other accounts
   - IP block → attacker rotates IPs

3. **Multiple defenses create real security**
   - Enterprise systems use many layers
   - If one fails, others catch the attack
   - This is defense-in-depth
   - This is why banks are secure
